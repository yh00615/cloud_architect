Amazon CloudFront란?

Amazon CloudFront는 전 세계에 분산된 엣지 로케이션을 통해 콘텐츠를 빠르게 전달하는 CDN(Content Delivery Network) 서비스입니다.

주요 기능

🚀 빠른 콘텐츠 전달

전 세계 450개 이상의 엣지 로케이션을 통해 낮은 지연 시간으로 콘텐츠를 제공합니다.

🔒 보안

AWS Shield Standard, SSL/TLS 암호화, OAC를 통한 S3 보안 접근을 제공합니다.

💰 비용 효율적

사용한 만큼만 지불하며, 오리진 서버의 부하를 줄여 비용을 절감합니다.

📊 실시간 분석

CloudWatch를 통해 실시간 메트릭과 로그를 확인할 수 있습니다.

캐싱 테스트

이 페이지는 CloudFront를 통해 전달되고 있습니다. 아래 정보를 확인하세요:

캐시 상태 확인

브라우저 개발자 도구(F12)를 열고 Network 탭에서 다음을 확인하세요:

  • X-Cache: Hit from cloudfront (캐시됨) 또는 Miss from cloudfront (캐시 미스)
  • X-Amz-Cf-Pop: 요청을 처리한 엣지 로케이션
  • X-Amz-Cf-Id: CloudFront 요청 ID

캐시 무효화 (Invalidation)

CloudFront 콘솔에서 캐시를 무효화하여 즉시 새 콘텐츠를 배포할 수 있습니다.

# AWS CLI로 캐시 무효화
aws cloudfront create-invalidation \
  --distribution-id YOUR_DISTRIBUTION_ID \
  --paths "/*"
