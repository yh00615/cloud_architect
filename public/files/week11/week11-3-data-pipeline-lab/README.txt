# Week 11-3 데이터 파이프라인 구축 실습 파일

이 ZIP 파일에는 실습에 필요한 샘플 데이터 파일이 포함되어 있습니다.

## 포함된 파일

1. sales-data.csv
   - 태스크 2에서 S3에 업로드할 초기 판매 데이터
   - 10개의 주문 데이터 포함

2. sales-data-2.csv
   - 태스크 10에서 파이프라인 테스트용 추가 데이터
   - 2개의 주문 데이터 포함

## 사용 방법

### 태스크 2: 샘플 데이터 업로드
1. sales-data.csv 파일을 S3의 raw-data 폴더에 업로드합니다.

### 태스크 10: 파이프라인 테스트
1. sales-data-2.csv 파일을 S3의 raw-data 폴더에 업로드합니다.
2. Lambda 함수가 자동으로 실행되어 데이터를 처리합니다.

## 주의사항

- CSV 파일은 UTF-8 인코딩으로 저장되어 있습니다.
- 첫 번째 줄은 헤더(컬럼명)입니다.
- 가격은 원(KRW) 단위입니다.
