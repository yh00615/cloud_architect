AWS CI/CD Pipeline에 대해 알아보기

이 프로젝트는?

Week 9-3 실습에서는 AWS Developer Tools를 활용하여 완전 자동화된 CI/CD 파이프라인을 구축합니다.

코드를 푸시하면 자동으로 빌드되고 S3에 배포되어 웹사이트가 업데이트됩니다.

사용된 AWS 서비스

  • AWS CodeCommit: Git 기반 소스 코드 저장소
  • AWS CodeBuild: 완전 관리형 빌드 서비스
  • AWS CodePipeline: CI/CD 파이프라인 오케스트레이션
  • Amazon S3: 정적 웹사이트 호스팅
  • Amazon EventBridge: 이벤트 기반 트리거

학습 목표

  • CodePipeline을 사용하여 S3 정적 웹사이트 배포 파이프라인을 구축할 수 있습니다
  • CodeCommit에서 소스 코드를 관리하고 변경 사항을 추적할 수 있습니다
  • CodeBuild로 정적 웹사이트를 빌드하고 S3에 배포할 수 있습니다
  • 코드 변경 시 자동으로 S3에 배포되는 CI/CD 워크플로우를 구현할 수 있습니다
