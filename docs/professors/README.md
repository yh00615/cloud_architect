# 교수용 가이드 안내

> 상황에 맞는 가이드를 선택하세요

## 📚 가이드 선택하기

### 🚀 처음 시작하시나요?

**[빠른 시작 가이드](QUICK_START_FOR_PROFESSORS.md)** 를 먼저 읽어보세요.
- ⏱️ 5분 안에 실습 가이드 수정 시작
- 🎯 필수 도구 및 기본 작업 방법
- 💡 자주 하는 작업 예시
- 🔧 일반적인 문제 해결

**이런 분들께 추천합니다:**
- ✅ 처음으로 실습 가이드를 수정하시는 분
- ✅ 빠르게 시작하고 싶으신 분
- ✅ 기본적인 수정만 필요하신 분

---

### ✏️ 실습 가이드를 수정하시나요?

**[교수용 가이드](PROFESSOR_GUIDE.md)** 를 참고하세요.
- 📝 마크다운 문법 전체 설명
- 🎨 AWS 버튼, Alert, 코드 블록 등 상세 문법
- 📋 실습 가이드 vs 데모 가이드 차이
- 🖼️ 스크린샷 추가 방법
- ✅ 작성 전 체크리스트
- 🎯 품질 기준 및 베스트 프랙티스

**이런 분들께 추천합니다:**
- ✅ 새로운 실습 가이드를 작성하시는 분
- ✅ 복잡한 마크다운 문법이 필요하신 분
- ✅ 스크린샷을 추가하고 싶으신 분
- ✅ 실습 가이드 품질을 높이고 싶으신 분

## 💡 추천 학습 경로

### 1단계: 빠른 시작 (필수)
1. **[빠른 시작 가이드](QUICK_START_FOR_PROFESSORS.md)** 읽기 (5분)
2. 간단한 수정 실습 (제목 변경, 단계 추가)
3. Git으로 저장 및 확인

### 2단계: 심화 학습 (선택)
1. **[교수용 가이드](PROFESSOR_GUIDE.md)** 의 "마크다운 문법" 섹션 읽기
2. Alert 박스, AWS 버튼 등 고급 문법 실습
3. 스크린샷 추가 실습

### 3단계: 전문가 (선택)
1. **[교수용 가이드](PROFESSOR_GUIDE.md)** 전체 읽기
2. 새로운 실습 가이드 작성
3. 품질 체크리스트로 검증
